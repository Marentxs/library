# Library Project

A simple JavaScript project that simulates a personal library where you can add, remove, and display books.
This project is designed to demonstrate object manipulation, arrays, and method usage in JavaScript.

# Features

- Add Books and remove books, create/delete book objects dynamically from your collection.

- View all books currently in the collection.

- Temporary Storage, the library data is stored only in memory (not persistent). Refreshing the page clears it.

# Built with

HTML5
CSS3 (Styling)
JavaScript (Logic and interactivity)
